- 基本需求
  - HTTPS 过滤 (需要ROOT)
  - 对第三方应用过滤 (购买正版)
- 如何导入
  - AdGuard 设置
  - 内容拦截
  - 过滤器
  - 自定义过滤器
  - 添加
  - 填入链接
